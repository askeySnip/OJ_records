Problem Statement

    

Consider all multiples of a positive integer M that do not exceed U. If we write each of them in base B, how many of them don't contain any zeros? Return the answer modulo 10^9 + 7.

Definition

    
Class:NiceMultiples
Method:count
Parameters:long long, long long, int
Returns:int
Method signature:int count(long long M, long long U, int B)
(be sure your method is public)

Limits

    
Time limit (s):2.000
Memory limit (MB):256

Constraints

-M will be between 1 and 10^12, inclusive.
-U will be between M and 10^12, inclusive.
-B will be between 2 and 16, inclusive.
